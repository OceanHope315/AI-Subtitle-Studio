# 更新日志

本项目的重要变更记录在此文件中，版本号遵循语义化版本规范。

## [1.1.0] - 2026-07-24

### 新增

- 新建任务时可选择“纯音频模式”或“音频 + 视觉模式”。
- 纯音频任务会跳过 ROI、视觉 OCR 和视觉进度事件，直接运行 WhisperX。
- 任务 API 新增 `analysis_mode` 字段，并使用 `skipped` 明确表示未启用的视觉来源。

### 修复

- 编辑页视频会按原始宽高比完整居中展示，不同尺寸的视频不再被裁切。
- 纯音频任务的进度页和编辑器不再显示无关的视觉识别内容。

### 兼容性

- 未提供 `analysis_mode` 的旧客户端和旧任务继续按“音频 + 视觉模式”处理。
